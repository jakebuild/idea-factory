## Description
an app that guide user how to make their phone ugly (screen, background, screen time limit, contract, color, font size...) then they will reduce their addictions to their phone and focus on working

### Critique
**Verdict: WORTH BUILDING — Score: 6/10**

The core insight is research-backed (grayscale, ugly wallpapers, removing color genuinely reduce dopamine hooks), and the "guided setup wizard" framing is smart — you're teaching users what to change, not enforcing it, which sidesteps OS permission nightmares. No backend needed, fully offline, one-time setup wizard. Doable solo in 2-4 weeks.

Key weaknesses: this is essentially a checklist/blog post dressed as an app with zero replayability — once users follow the steps, they never return. Android vs iOS fragmentation will silently eat your time. The "contract" concept is vague. Competitors like iOS Screen Time and Digital Wellbeing already do this for free. Business model is unclear for a one-time-use app.

Strongest path forward: iPhone-first only, add a daily dashboard showing screen time trend (the retention hook), a paid one-time $2.99 purchase, and a before/after ugly home screen share feature for organic growth.

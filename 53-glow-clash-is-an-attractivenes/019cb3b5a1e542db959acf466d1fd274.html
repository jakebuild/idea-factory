MULTI-TASKING PRODUCTS

Even if one product serves multiple functions, select all relevant categories. A single formula can contain both chemical exfoliants and retinoids—both must be logged to calculate the interaction density accurately.

02
CONSERVATIVE GUIDANCE

The algorithm defaults to maximum safety. If a molecular clash is mathematically possible, the system will flag it as HIGH RISK.
